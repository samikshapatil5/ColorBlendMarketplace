# ColorBlendMarketplace 🎨

An Android-based marketplace application for interior design and painting services.

## 📱 Features
- User authentication
- Admin module
- Interior design module
- Painter management
- Order management
- Firebase integration

## 🛠 Tech Stack
- Java
- XML
- Android Studio
- Firebase
- Gradle (KTS)

## 📦 Project Structure
- app/
- admin/
- interiordesign/
- ordermanage/
- painter/
- visualizing/

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Device
